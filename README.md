> Quasar Crud Component based on QTable using Axios API to fetch remote data.

## Links
* [Vuejs Page](https://vuejs.org/)
* [Quasar Framework Page](http://quasar-framework.org/)

## Install 
```
quasar ext add crud
```

## Uninstall
```
quasar ext remove crud
```
